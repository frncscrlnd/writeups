# [No Alphabets and Digits](https://xss.challenge.training.hacq.me/challenges/easy01.php)

Let's take a look at this page's source code:

```
<script src="hook.js"></script>
<?php
// by escaping the payload you won't break this system, haha! :-)
$escaped = preg_replace("/[a-zA-Z0-9]/", "", $_GET['payload']);
?>

<script>
    // here you can inject an arbitrary script,
    // but I guess you can't do anything, cuz the script can't include a-zA-Z0-9 ! :-)
    <?= $escaped ?>
</script>


<h1>inject</h1>
<form>
    <input type="text" name="payload" placeholder="your payload here">
    <input type="submit" value="GO">
</form>

<h1>src</h1>
<?php highlight_string(file_get_contents(basename(__FILE__))); ?>
```

Just like we've previosly seen in [Baby XSS 04](https://frncscrlnd.github.io/writeups/y0n3uchy_xss/Baby-XSS-04/), we come across [`preg_replace`](https://www.php.net/manual/en/function.preg-replace.php), a sanitization function that will search and replace regular expressions. In our case this function will remove (replace with an empty string `""`) all alphabetical characters, both lowercase and uppercase, and all digits from 0 to 9 from our payload:

```
$escaped = preg_replace("/[a-zA-Z0-9]/", "", $_GET['payload']);
```

It is most definitely impossibile to submit a "normal" payload without using these characters. We can however, use [JSFuck](https://jsfuck.com/), a JavaScript programming style inspired by [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck) that only uses six characters: `[`, `]`, `(`, `)`, `!`, and `+`.
